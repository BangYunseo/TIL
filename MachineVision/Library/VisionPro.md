## 3절. 라이브러리

### PatMax

- VisionPro의 대표적인 기술
- 기하학적 패턴 매칭
- 픽셀 그리드가 아닌 경계 곡선 집합을 기반으로 물체의 형상 학습
- 각도, 크기, 조명 변화에도 높은 정확도로 물체 식별
- 자동차·전자·반도체·식품·제약 등 5대 주요 산업에서 활용
  - 부품 정렬
  - 조립 검증
  - 품질 검사

### OCRMax

- 광학 문자 인식 도구
- 판독이 까다로운 문자에 대해 높은 정확도 제공
- 자동 조절 기능으로 최적의 세분화 파라미터 설정
- 글꼴 자동 학습 후 설치 시간 단축
- 활용 범위
  - 대비가 낮은 환경
  - 왜곡된 텍스트
  - 비례 글꼴
  - 가변 문자열 길이

### IDMax

- 바코드 리딩 기술
- 1D / 2D 코드 동시 판독 가능
- 이력 추적 및 제품 식별
  - 물류 산업
  - 제조 산업
  - 제약 산업
