## 3절. 에디터
#### 키보드

|키|내용|
|:---:|:---|
|Q|뷰 이동|
|W|오브젝트 이동|
|E|오브젝트 회전|
|R|오브젝트 크기 조절|
|T|사각툴(2D, UI 오브젝트 크기 조절)|
|마우스 우클릭|카메라 회전|
|alt + 마우스 좌클릭|카메라 축 이동|
|키보드 상하좌우 방향키|카메라 자유 이동|
|마우스 휠|카메라 이동, 줌|


## 4절. 게임 구조
#### 게임과 연극의 관련성

- 연극 == 게임
- 막 == 장면
- 인물 & 소품 == 오브젝트
- 대본 == 스크립트

#### 프로그래밍 언어

- 컴퓨터 언어로 나누는 대화

    - 컴퓨터 : 기계어
    - 사람 : 프로그래밍 언어

- Unity : C# 사용
