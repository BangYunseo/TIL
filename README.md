# TIL - Today I Learned

## 파일 생성 예시

- [파일명].[md]
- 파일명은 숫자 제외 요약 항목 명칭
  - ex : API.md, MVVM.md, UI.md 등
- md 파일 고정
- 이미지는 폴더 내 Image에 파일명과 동일한 폴더 생성 후 저장

## 컨벤션 예시

### Git Commit Type

- feat: 새로운 기능 추가
- fix: 버그 수정
- docs: 문서 수정
- style: 코드 스타일 변경 (코드 포매팅, 세미콜론 누락 등)
- design: 사용자 UI 디자인 변경
- test: 테스트 코드, 리팩토링
- refactor: 리팩토링
- build: 빌드 파일 수정
- ci: CI 설정 파일 수정
- perf: 성능 개선
- chore: 자잘한 수정이나 빌드 업데이트
- rename: 파일 혹은 폴더명을 수정만 한 경우
- remove: 파일을 삭제만 한 경우
