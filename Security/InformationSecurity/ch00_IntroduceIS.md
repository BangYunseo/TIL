## 1절. What is Security?
#### 컴퓨터 보안
* 정보 시스템 자원(HW / SW / 펌웨어 / 정보, 데이터 / 통신)에 대한 무결성과 가용성, 기밀성 유지와 같은 목적 달성을 위해 자동화된 정보 시스템에 적용되는 보호
    * NIST Computer Security HandBook에서의 정의

#### 보안 목적
![CIA](https://)
* 보안의 3요소
    * Confidentiality : 기밀성
    * Integrity : 무결성
    * Availability : 가용성

* 추가적인 목적
    * Authenticity : 진실성
    * Accountability : 대처능력
