## 2절. 배정문

### 정의

- 값을 변수에 저장할 경우 사용

### 형태

```C#
// 변수 할당 예시

remainder = dividend % divisor;
i = j = k = 0;
var op = exp;
```

1. <변수> = <식>;

   - C언어와 동일한 형태

2. a = b = exp;

   - 좌측 부분이 여러 번 할당될 경우 exp 먼저 연산
   - 그 후 b에 할당하고 다시 b 값이 a에 할당

3. 초깃값 할당
   - 변수 초깃값 할당 시 유용
