## 0절. 개요

### 그래프(Graph)

- 현상이나 사물을 정점과 간선으로 표현
- $Graph$ $G$ = $(V,E)$
  - $V$ : vertex set
  - $E$ : edge set
- 정점(Vertex) : 대상 or 개체
- 간선(Edge) : 정점 간 관계
- 인접(Adjacent) : 두 정점이 간선으로 연결된 경우
