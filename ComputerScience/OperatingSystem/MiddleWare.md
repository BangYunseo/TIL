## 1절. MiddleWare

### MiddleWare

- <strong>양 쪽을 연결</strong>해 데이터를 주고 받을 수 있도록 <strong>중간에서 매개 역할</strong>을 하는 소프트웨어
- 서로 다른 애플리케이션이 통신할 때 사용되는 소프트웨어
- 애플리케이션을 지능적이고 효율적으로 연결하는 기능 제공
- 단일 시스템에 원활하게 통합할 수 있도록 다양한 기술, 도구, 데이터베이스 간 다리 역할
