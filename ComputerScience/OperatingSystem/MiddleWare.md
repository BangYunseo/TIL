### 서버 유형

|                  유형                  | 특성                                                                                                              | 예시                                                   |
| :------------------------------------: | :---------------------------------------------------------------------------------------------------------------- | :----------------------------------------------------- |
|               WEB Server               | 사용자의 요청이 유입되면 호출되는 앞단(Front-End)의 정적 페이지(HTML, CSS, JS, PNG 등)를 전용으로 처리하는 서버   | Apache, Microsoft IIS, 티맥스소프트 WEBTOBE, NGINX etc |
| WAS Server<br>(Web Application Server) | 로그인, 검색 등 데이터를 가공하고 처리하는 뒷단(Back-End)의 동적 페이지(jsp, servlet 등)를 전용으로 처리하는 서버 | Tomcat, 웹 로직, 웹 스피어, 제우스                     |

## 1절. 미들웨어

### 미들웨어(MiddleWare)

- <strong>양 쪽을 연결</strong>해 데이터를 주고 받을 수 있도록 <strong>중간에서 매개 역할</strong>을 하는 소프트웨어
- 서버의 요청(Request) <=> 응답(Response) 과정의 중간에 위치해 특정 기능을 수행하는 함수

> 즉, 통상적으로 <strong>미들웨어 == WAS 서버</strong>이다.

### 미들웨어 종류

#### 오픈소스

|     종류      | 특성                                                                                                                                                                                                                                 |
| :-----------: | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Apache Tomcat | - Apache 재단에서 제공하는 Java 기반의 웹 애플리케이션 서버<br>- 웹 서버와 서블릿 컨테이너 제공<br>- 오픈소스이므로 무료로 사용 가능<br>- 오랫동안 릴리즈 진행<br>- 많이 사용하는 솔루션으로 다수의 레퍼런스 존재<br>- 벤더의 지원 X |
|    WildFly    | - JBoss.org에서 제공하는 오픈소스 WAS                                                                                                                                                                                                |

#### 상용

|   종류    | 특성                                                                                                                                                             |
| :-------: | :--------------------------------------------------------------------------------------------------------------------------------------------------------------- |
|   JEUS    | - 국내 기업인 TmaxSoft에서 제공하는 상용 WAS<br>- Tomcat과 유사한 구조로 관공서 및 국내 공공 기관에서 많이 사용하는 솔루션<br>- 국내 기업으로 빠른 기술지원 제공 |
| JBoss-EAP | - RedHat에서 제공하는 WAS<br>- WildFly의 상용버전<br>- 외국계 상용 WAS 중 JBoss가 국내에서 인기<br>                                                              |
